# Stable Diffusion XL (SDXL) Prompt Guide
- **Structure**: (Subject:1.5), (Action), (Context), (Art Style), (Tags)
- **Preferences**:
  - Heavy use of Danbooru tags and weighting syntax (e.g., (keyword:1.2)).
  - Separate concepts with commas.
  - Keywords: "masterpiece", "best quality", "8k", "HDR", "intricate details".
  - Negative Prompt is crucial.
