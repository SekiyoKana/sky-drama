# Midjourney V6 Prompt Guide
- **Structure**: [Subject] + [Environment] + [Lighting/Color] + [Mood] + [Composition] + --v 6.0
- **Preferences**:
  - Prefer natural language sentences over tag soups.
  - Use "shot on [camera]" for photorealism.
  - Avoid negative prompts (use --no parameter instead).
  - Keywords: "hyper-realistic", "cinematic", "detailed texture", "volumetric lighting".
