# Nanobanana Model Style Guide

Nanobanana is a high-fidelity specialized model. Prompts must follow a specific sequence to trigger its best performance.

- **Structure**: `[Trigger Word] + [Subject] + [Environment] + [Lighting/Mood] + [Composition] + [Negative Prompt]`
- **Trigger Word**: Always start with **`nanostyle`** (or your specific trigger).

## Key Parameters
- **Art Style**: "highly detailed", "sharp focus", "vibrant colors", "smooth texture".
- **Lighting**: "volumetric lighting", "studio soft light", "rim light".
- **Negative**: "low quality, blurry, distorted, ugly, text, watermark".
